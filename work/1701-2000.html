2 :: ‎ Record ID היא תכונת מערכת ספציפית במיוחד ל EFES CRM. מאחר וכך, כדי שקובץ הייבוא שלך יכלול את השדה Record ID של EFES CRM חובה לייצא קודם את הרשומות מ EFES CRM‎.
3 :: להלן תרחיש לדוגמה: אתה מייצא לידים מ EFES CRM בגיליון אלקטרוני, מבצע שינויים בכמה רשומות ומייבא אותן חזרה ל EFES CRM. במקרה כזה יהיה מתאים לבחור את השדה Lead ID כשדה הייחודי, מאחר ואתה רק מייבא חזרה לידים שיוצאו מ EFES CRM מלכתחילה. בצורה זו EFES CRM מחפש IDs תואמים של לידים ומעדכן את הרשומות התואמות.
